## 🚨 常見問題快速檢查

### 如果後端啟動失敗：
- [ ] 檢查 `firebase-service-account.json` 檔案是否存在
- [ ] 檢查 `.env` 檔案中的路徑是否正確
- [ ] 檢查 JSON 檔案格式是否正確

### 如果前端無法連接 Firebase：
- [ ] 檢查 `frontend/.env` 檔案是否存在
- [ ] 檢查所有 `VITE_` 開頭的環境變數是否已設定
- [ ] 重新啟動前端開發伺服器

### 如果註冊/登入失敗：
- [ ] 確認 Firebase Console 中 Email/Password 已啟用
- [ ] 檢查瀏覽器控制台的錯誤資訊
- [ ] 確認後端已啟動
